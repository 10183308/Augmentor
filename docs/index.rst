Augmentor
=========

Augmentor is a Python package designed to aid the artificial generation of image data for machine learning tasks. It is primarily a data augmentation tool, but will also incorporate basic image preprocessing functionality.

.. note:: 

    A Julia version of the package is currrently being developed. You can find it `here <https://github.com/Evizero/Augmentor.jl>`_.
